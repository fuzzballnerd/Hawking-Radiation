## About

This repository accompanies my computational project on Hawking radiation
and Primordial Black Hole (PBH) evaporation at IISc. The project builds
from classical black hole solutions up through semi-classical gravity to
numerical simulation of PBH evaporation, covering:

- **Classical black hole solutions** — Schwarzschild and Kerr metrics,
  conformal diagrams, and spacetime structure near the event horizon
- **Semi-classical gravity** — laws of black hole mechanics, derivation of
  Hawking temperature and Bekenstein-Hawking entropy
- **Numerical simulation** of PBH evaporation using BlackHawk, computing
  greybody factors to model particle emission spectra and mass evolution

**Primary tools:** BlackHawk, Python (NumPy, SciPy), Mathematica
